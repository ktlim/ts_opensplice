Detailed Description

QueryCondition objects are specialized ReadCondition objects that allow the application to also specify a filter on the locally available data.

The query (query_expression) is similar to an SQL WHERE clause can be parameterized by arguments.

Precise syntax for the query expression can be found in Annex A.
