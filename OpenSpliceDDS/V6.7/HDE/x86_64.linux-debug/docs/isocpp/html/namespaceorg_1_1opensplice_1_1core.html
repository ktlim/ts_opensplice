std::string org::opensplice::core::exception_helper ( const std::string &  message,
bool  ospl_error_info = true,
bool  stack_trace = true 
)

Produce a string suitable for populating an exception 'what' field. E.g:

std::string message = "dds::core::InvalidDataError";
message += context_to_string(OSPL_CONTEXT_LITERAL(""));
message += " Something bad happened. Some more information."
throw dds::core::NullReferenceError(
org::opensplice::core::exception_helper(message, false));

The above example shows how to produce an arbitrary-ish string. Note the non-default false second ospl_error_info argument that switches off the inclusion of the preceding erro info. This is appropriate if the error has not resulted from an underlying 'DCPS' call.
