Detailed Description

This file provides definitions for stdint header types for versions of Windows that didn't have it. It was discovered on the intahwebs at http://snipplr.com/view/18199/stdinth/ where it was described thus: "A public domain stdint.h. Original code is from a public domain version in the MinGW distribution. This one has been modified to fix a couple bugs (to make some macros 64-bit aware) and to make it compatible with VC6." It was published there on 08/12/09
